# Mergesort
## Goal:
Combine two arrays and sort them using mergesort.

## Input:
two numbers `n` and `m` and then `n` numbers and `m` numbers, being the two arrays.

## Output:
It outputs sorted merged array and which numbers are from which array. <br>
a-first, b-second

Sample output is:
```
babab
153 527 741 808 879 
```
